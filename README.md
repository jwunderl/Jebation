# Jebation

Script to transform image or gif into rainbow gif.

install deps:

```
python3 -m pip install --upgrade pip
python3 -m pip install --upgrade Pillow
python3 -m pip install --upgrade numpy
```

Do `python3 jebation.py [path]` for use it.
